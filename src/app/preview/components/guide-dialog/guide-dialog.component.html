Guide

An interactive area defines a region on the viewport that reacts to a gesture performed by the user by animating a portion of the spritesheet.
To define an interactive area, click on the Add interactive area button, select the shape the area should have and draw it on the sprite sheet.
Once the area has been drawn, its set up can be completed by specifying:
  • the gesture the area should react to
  • the start frame from which the sprite sheet shall start its transition
  • the end frame where such a transition shall stop
  • the direction, horizontal or vertical, in which the sprite shall slide
By default, the area is automatically disabled once the spritesheet reaches the end frame, meaning that it will stop reacting to the user gestures until the spritesheet is reset to the start frame. To change this behavior, untick from the checkbox Disable area on end.
