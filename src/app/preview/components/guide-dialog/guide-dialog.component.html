Guide - Interactive Areas

An interactive area defines a region on the viewport that reacts to a gesture performed by the user by animating a portion of the sprite-sheet.
To define an interactive area, click on the Add interactive area button, select the shape the area should have and draw it on the sprite-sheet.
Once the area has been drawn, its configuration can be completed by specifying:
  • the gesture the area should react to
  • the direction, horizontal or vertical, in which the sprite-sheet shall slide to animate
  • the start frame from which the animation shall start. This frame can be changed by dragging the shape onto another tile of the sprite-sheet
  • the end frame where the animation shall stop
  • whether the area shall also behave as undo action
By default, the area is not disabled once the animation reaches the end frame. Therefore, performing the gesture again will revert the animation to the initial frame. To change this behavior and define a custom undo action, deselect the checkbox Use this area also for the undo action.
